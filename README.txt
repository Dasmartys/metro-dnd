
# Metro DnD Bestiář

## Jak přidat novou kartu:

1. Přidej obrázek do složky `img/`. (Např. img/novy_enemy.jpg)
2. Otevři soubor `index.html`.
3. Najdi sekci `const data = [ ... ]` a přidej nový záznam podle vzoru.
4. Ulož a nahraj změny na GitHub.

## Jak nahrát na GitHub Pages

1. Vytvoř si účet na https://github.com a nový repozitář.
2. Nahraj všechny soubory (včetně složky img a index.html).
3. V nastavení repozitáře zapni GitHub Pages -> Deploy from branch -> main -> root.
4. Hotovo! Otevři vygenerovanou adresu.

Např: https://tvujlogin.github.io/metro-dnd-bestiary
